Prediksi Nilai Tukar USD

Proyek ini bertujuan untuk memprediksi nilai tukar USD terhadap Rupiah menggunakan metode Artificial Neural Network (ANN) berbasis feedforward neural network sederhana. Model dikembangkan menggunakan Python, Keras, dan Scikit-Learn, dengan data historis kurs transaksi Bank Indonesia selama satu tahun terakhir. Melalui proses normalisasi, pelatihan, dan prediksi iteratif, sistem ini mampu memberikan estimasi nilai tukar selama 30 hari ke depan berdasarkan pola tren data sebelumnya.

Prediksi Nilai Tukar USD

Project Overview

Data yang saya gunakan bersumber dari data Bank Indonesia "https://www.bi.go.id/id/statistik/informasi-kurs/transaksi-bi/default.aspx" dalam periode satu tahun (1 Desember 2023 - 29 November 2024)

Key Highlights

    Model berhasil menghasilkan prediksi tren nilai tukar harian selama 30 hari ke depan. Hasil prediksi disajikan dalam tabel berikut (contoh hasil simulasi):

Tekhnologi Yang Digunakan :

Bahasa Pemrograman : Python

Library Utama : Pandas & NumPy, Scikit-Learn, Keras (TensorFlow backend)

Dataset : File Excel berisi data historis nilai tukar USD selama 1 tahun

Tahapan Pengembangan

1. Data nilai tukar dibaca menggunakan pandas, kemudian dilakukan normalisasi ke rentang (0 – 1) agar model lebih stabil dalam proses pelatihan.

data = pd.read_excel('Kurs Transaksi USD 1 tahun.xlsx', nrows=250)
scaler = MinMaxScaler(feature_range=(0, 1))
data['Nilai'] = scaler.fit_transform(data['Nilai'].values.reshape(-1, 1))

2. Pembuatan Dataset Pelatihan :
Dataset dibagi menjadi input dan target
x: Nilai Tukar Hari ini
y: nilai tukar hari berikutnya

X = data['Nilai'].values[:-1].reshape(-1, 1)
y = data['Nilai'].values[1:]

Perancangan Model ANN :
Model jaringan saraf terdiri dari:
1 neuron input
2 hidden layer (10 dan 5 neuron) dengan aktivasi ReLU
1 output neuron tanpa aktivasi (regresi linear)

model = Sequential()
model.add(Dense(10, input_dim=1, activation='relu'))
model.add(Dense(5, activation='relu'))
model.add(Dense(1))
model.compile(loss='mean_squared_error', optimizer='adam')

4. Proses Pelatihan :
Model dilatih sebanyak 100 epoch dengan ukuran batch 10.

model.fit(X, y, epochs=100, batch_size=10, verbose=1)

Prediksi Nilai Tukar :
Model digunakan untuk memprediksi nilai tukar selama 30 hari ke depan secara iteratif.

predictions = []
last_value = data['Nilai'].values[-1]

for _ in range(30):
next_value = model.predict(np.array([[last_value]]))
predictions.append(scaler.inverse_transform(next_value)[0][0])
last_value = next_value[0][0]
